About the project

Where metallurgy meets open data

Metalúrgica Data is an open initiative to build a public knowledge base of metallurgical process data — synthetic, educational, and official. Built for students, engineers, and data scientists who want to apply ML and analytics to the mining industry.

All datasets are documented with variable descriptions, units, and technical context. Each file is ready to load in Python, Power BI, or any data tool.

Contribute

Help build the collection

This project grows with community contributions. Whether you have real anonymized data, synthetic datasets, or analytical notebooks — your input is welcome.

Read the guide
  • 1
    Fork the repository and create a new branch for your dataset.
  • 2
    Add your CSV files inside the appropriate datasets/ subfolder.
  • 3
    Include a README.md with variable descriptions, units, and source information.
  • 4
    Open a Pull Request — we review and merge promptly.
  • 5
    Optionally share an analysis notebook in notebooks/.
